About (in the form of an FAQ)

WTF am I looking at?

Mass Effect 5e is a homebrew conversion of Dungeons and Dragons 5th Edition for the Mass Effect Universe.

How do I use it?

Prerequisite: You've played D&D 5e. If not, get a player's handbook and read those rules first. This system builds on the underlying game architecture of D&D 5e.

  1. Read the rule changes and additions for this system Rules
  2. The rest of the site acts as you player's handbook:
    • Races
    • Classes
    • Features (a.k.a. spells)
    • Weapons
    • Heavy Weapons

What is the status of this project?

Follow the news section for updates.

How can I contribute?

In a few ways:

  1. The easiest way is to submit a Github issue. This could be something as simple as Hey! You misspelled 'biotic' in the text of 'Biotic Mastery' feature. Or it could be a request, like Please add armor!!!
  2. Additionally, you can comment on the Google Sheet.
  3. If you're so inclined, you can even clone the repo and make changes, then submit a pull request.
